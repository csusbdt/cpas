Preparation

The code presented in this activity requires Python3.

Study or review the material in the following 2 tutorials.

  • HTML Dog's HTML Beginner Tutorial.
  • HTML Dog's CSS Beginner Tutorial.

Instructions

Create a simple Website and publish it to the public Internet. Either figure this out your own way or complete Experiment with Web Publishing. (1 week)

Instead of creating your site's HTML files manually, write a Python script that generates them from files containing textual data in Markdown syntax. To do this, complete Experiment with Markdown. (1 week)

Style your website using a CSS framework. Either do this on your own or complete Experiment with CSS. Continue to develop and use your website generation script. (1 week)

Automate the entire process of transforming markdown into published HTML. Complete Experiment with Sockets. Warning: advanced. (1 week)
