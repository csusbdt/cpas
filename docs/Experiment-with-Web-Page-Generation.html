Preparation (1 week)

The code presented in this activity requires Python3.

Study or review the material in the following 2 tutorials.

  • HTML Dog's HTML Beginner Tutorial.
  • HTML Dog's CSS Beginner Tutorial.

Read the Wikipedia entry on Markdown.

Options

To generate a website from textual data in Markdown syntax, complete an Experiment with Markdown. (1 week)

To style your markdown-based site using a CSS framework, complete Experiment with CSS. (1 week and requires completion of Experiment with Markdown)

To publish your website to the public Internet, complete Experiment with Web Publishing. (.5 weeks and requires completion of Experiment with Markdown)

To automate the transformation of markdown into published HTML, complete Experiment with Sockets. (.5 weeks and requires completion of Experiment with Web Publishing)
