From this point, I will explain to you how I solved this problem.

I installed the Python-based markdown conversion library Python-Markdown into my Python3 environment. To do this, I forked the original project to make one small modification: I have the wikilinks extension use dashes instead of underscores. I did this to better match the markdown supported by GitHub. I used the following to install my fork of the library as follows.

git clone git@github.com:csusbdt/Python-Markdown.git
cd Python-Markdown
sudo python3 setup.py install

Test the library with the following.

import markdown

md = markdown.Markdown([
    "markdown.extensions.wikilinks(base_url=, end_url=.html)",
    "markdown.extensions.tables",
    "markdown.extensions.fenced_code"]
)

text = """
I'm a __bold__ word in the first paragraph.

I'm a [[Wiki Link]] in the second paragraph.

I'm an [External Link](https://github.com/csusbdt/cpas/wiki) in the third paragraph.
"""

html = md.convert(text)
print(html)
