In this course, you need to learn 2 command line interfaces. You need to learn the command line interface for the console (or terminal) window application of your operating system. And you need to learn the command line interface of the Python interpreter.

Before starting this activity, see Install Python to make sure Python is installed on your computer.

At a minimum, you should learn the following UNIX commands for this course.

command description example
pwd print working directory pwd
cd change directory cd basics
cp copy file or directory cp ex2.cpp ex3.cpp
mv move file or directory mv hello.cpp bye.cpp
rm remove file or directory rm a.out
mkdir make directory mkdir basics
rmdir remove directory rmdir basics

You also need to know how to use the following special symbols inside file system path names.

symbol meaning example
. current directory ./a.out
.. parent directory cd ..
~ home directory rm ~/a.out

command completion

One helpful feature of the terminal window is called command completion. If you press tab in the command line, the system will try to complete what you are typing by looking for a matching filename.

command history

Another helpful feature is called command history. This feature allows you to access a record of commands that you previously entered. Use the up and down arrow keys to scroll through the command history. Press enter at any previously entered command to run it again.

The interactive Codecademy command line tutorial is a good place to learn about the Unix command line, although the tutorial covers topics beyond those required by this course. Whether you use this resource or not, you need to learn how the above Unix commands work in your system.

Instructions:

  1. In your home directory, create a folder named cse0101.
  2. Change the working directory to ~/cse0101.
  3. Use a text editor to create a file named hello.py under ~/cse0101 to contain the single line print "hi"
  4. Run the python script with the command python hi.py.
  5. Create a copy of hi.py named bye.py. Change bye.py so that it prints bye instead of hi.
  6. Create a subfolder under ~/cse0101 named lab1 and move hi.py and bye.py into it.
  7. Continue to practice file system commands by devising your own experiments.

Experience Working Python

Start the command line interpreter by entering python on the command line.

python

You can recognize the Python command line by the prompt >>>.

At the Python command line, enter the following mathematical expressions and observe what happens. (Do not type the prompt >>>.)

>>> 2 + 3
>>> 2 * 3
>>> 2 / 3
>>> 2 // 3
>>> 2 ** 3
>>> 2 % 3

Explain in writing what each of these expressions do.

Variables are labels that identify values. The assignment operator = changes the value assigned to a variable. The following lines of Python code contain variables. At the Python command line, enter these lines and observe what happens.

>>> width = 2
>>> height = 3
>>> width * height
>>> width = 4
>>> width * height

Explain in writing what each of these expressions do.

At the Python command line, enter the following string expressions and observe what happens.

>>> 'Alice says hi.'
>>> 'Alice says "hi."'
>>> 'Alice says ' + '"hi."'
>>> 'Bob\'s villa.'

Explain what the following expressions do.

s = 'Hello Bono.'
s[0]
s[1]
s[2]
s[0:4]
s[8:10]

At the Python command line, enter the following 2 lines and observe what happens.

"Dear John,\nGood bye.\nLove Alice"
print("Dear John,\nGood bye.\nLove Alice")

Explain what the character sequence \n represents.

Perform additional experiments to strengthen your understanding of strings.

Input an end-of-file character by entering Ctrl-D to exit the Python command line interpreter.

Be prepared to answer quiz questions on these activities.
