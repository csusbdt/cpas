Course title and description

CSE 1010 Introduction to Computer Programming

Students learn to code using a scripting language such as Python or Javascript. Language syntax, basic data types, control flow, functions, classes, data structures, command line and graphical user interfaces, application programming interfaces (APIs), automation experiments in one or more problem domains such as animation, simulation, web publishing, etc. (3 units)

Course activities

Do 10 weeks of core activities.

Core activities

  • Install Python (0 weeks)
  • Learn the Command Line Interface (1 week)
  • Learn the Python Language (9 weeks)

Do 5 weeks of elective activities.

Elective activities

  • Experiment with Web Page Generation (5 weeks)
  • Experiment with 3D (5 weeks)
  • Experiment with Simulation (5 weeks)

Pedagogy

This course will use writing to support learning in the following ways.

  • Use writing to support the attainment of skills. A learning history is a written record of what you read, thought about and experimented with when trying to develop a new skill needed to solve a problem. (Develop an example.)
  • Use writing to support problem solving tasks. Students write documentation as part of the process of solving computer programming problems. (Develop an example.)

Apply the scientific method to personal learning tasks. Easy-to-knock-off experiments tempt the learner into guessing. However, experiments should lead to improvements in conceptual understanding. To gain this benefit, students need to formulate clear hypotheses based on their conceptual understanding before running computer programming experiments. If the outcome of the experiment is contrary to the prediction, the student needs to adjust their conceptual models.

Think about and discuss the creative process of learning.

All lectures are videos that can be viewed at any time. On-campus meetings of 3 hours per week are activity sessions in which students focus on problem solving, collaboration, research and programming experiments. It is also time for examinations and general discussion.

Work on assignments is submitted through sharable workspaces such as Cloud9 or a version control system such as Git.

Grading

Grades will be based on assignments and written exams.

General Education Learning Outcomes

This course is being designed to satisfy the A3 Critical Thinking category of the general education program. The GE learning outcomes include the following.

  • Learning How to Learn/Metacognition - Develop awareness of their learning processes, becoming reflective, self-directed learners who are able to apply and adapt their processes of learning in new contexts.
  • Thinking Critically - Think critically, evaluate, analyze, and solve problems employing multiple methods of reasoning.
  • Critical Literacies - Analyze the ways that information, including quantitative and technological, oral and written, both shape and are shaped by social contexts.
  • Ethical Responsibility - Recognize that they are ethically responsible for the impact that their ideas, decisions, and actions have upon their lives and local and global communities.
