Course title and description

CSE 1010 Computer Programming in Context

CSE 1010 Introduction to Computer Programming

Students learn to code using a scripting language such as Python or Javascript. Language syntax, basic data types, control flow, functions, classes, data structures, command line and graphical user interfaces, application programming interfaces (APIs), automation experiments in one or more contexts such as animation, simulation, Web publishing, etc. (3 units)

Course activities

Do 10 weeks of core activities.

Core activities

  • Install Python (0 weeks)
  • Learn the Command Line Interface (1 week)
  • Learn the Python Language (9 weeks)

Do 5 weeks of elective activities.

Elective activities

  • Experiment with Web Page Generation (3 weeks)
  • Experiment with 3D (3 weeks)
  • Experiment with Simulation (1 week)

Pedagogy

Use writing to support learning.

  • Use note taking to support the attainment of skills. A learning history is a written record of what you read, thought about and experimented with when trying to develop a new skill needed to solve a problem. (Develop an example.)
  • Use writing to support problem solving tasks. Students write documentation as part of the process of solving computer programming problems. (Develop an example.)
  • Use writing to support the attainment of skills. (Example)

Apply the scientific method to personal learning tasks. Students formulate clear hypotheses before running computer programming experiments. The results of programming experiments are clearly predicted prior to running the experiment. Easy-to-knock-off experiments tempt the learner into shallow guessing; experiments should lead to improvements in conceptual understanding within a given problem domain.

Teach how to learn. This is easy: just show how to do it. Learn something new with the class. Demonstrate the creative process of learning.
