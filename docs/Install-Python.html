OS X and Linux computers

Later version of Mac OS X and many distributions of Linux include Python pre-installed. However, it's a good idea to check that Python is installed and ready for use. To do this, open a command line window, also call a terminal window, and enter the command to run the Python command line interpreter as follows.

python

If it leaves you at the Python command line, signified by 3 greater than symbols, then Python is installed. You can exit the command line interpreter by holding down the control key and pressing the D key. (This inserts an end of file character into the command line's input stream.)

If instead you receive a message saying command not found, or something to that effect, then Python is not installed on your system. See the downloads page of the Python Beginner's Guide for installation instructions.

How to install Python3 in Mac OS X to exist alongside pre-installed Python 2

The command line tools from Xcode are needed. If you do not have Xcode, then you should install it from the App Store.

After installing Xcode, run xcode-select --install in the terminal command line to install the command line tools.

You need to install Brew. Enter brew at the command line to see if you have Brew already installed. If the command isn't found, install it with the following command.

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Now, you can run python3 with the following command.

python3

See README.md for instructions on how to setup and use Python3 in a virtual environment, which you will need to do if you are using the lab machines or other machine on which you do not have administrator privileges.
