Blender Command Line

We can write to the console window from inside Python scripts.

Here is an example of using the command line to render an animation.

blender --background --enable-autoexec myanim.blend --render-anim

The --enable-autoexec option will run initialization scripts in myanim.blend.

NOTE: Argument order on the Blender command line is significant. See documentation on this.
